Trois joueurs avancent dans l’ombre sous la cape des Masqués. Le reste de l’équipe forme le camp des Loyaux. Chaque avis obtenu peut faire basculer la semaine. Règles du jeu

Une lecture rapide, claire et sans perte de temps.
Masqués
Il y a 3 Masqués dans le magasin. Un avis positif obtenu à leur nom vaut 2 points.
Loyaux
Tous les autres joueurs sont Loyaux. Un avis positif obtenu à leur nom vaut 1 point.
Après chaque avis
Le joueur choisit : sécuriser pour prendre +1 point garanti, ou tenter une épreuve.
Épreuves
Une épreuve réussie de manière parfaite rapporte +3 points. En cas d’échec, le joueur repart avec 0 point.
Enquêteur de la semaine
La personne ayant obtenu le plus d’avis positifs à son nom devient Enquêteur de la semaine.
Table ronde
Le lundi, l’Enquêteur peut ouvrir la table ronde et envoyer son accusation par WhatsApp. Une seule tentative de démasquage est autorisée.
Important : l’équipe peut se concerter avant le vote, mais les Masqués sont présents dans le groupe. Une mauvaise influence peut donc envoyer l’Enquêteur sur un faux suspect.

Table ronde

Ouverte uniquement le lundi. L’Enquêteur de la semaine peut envoyer son accusation.
La règle du lundi
La table ronde n’est ouverte qu’une fois par semaine. La personne ayant obtenu le plus d’avis positifs à son nom peut voter. Elle peut se concerter avec l’équipe avant d’envoyer le nom du suspect, mais les Masqués sont présents dans le groupe et peuvent pousser sur une mauvaise piste.
