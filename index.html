Mission

Trois Masqués avancent dans l’ombre. Les Loyaux doivent faire progresser leur camp, tenter les épreuves et réussir le bon démasquage en fin de semaine.
3 Masqués
1 tentative de démasquage / semaine
+3 points si épreuve réussie

Enquêteur
À venir
Les Loyaux gardent l’avantage pour l’instant, mais une seule série d’épreuves réussies peut faire basculer la semaine.

Joueur

Le prénom sera injecté automatiquement dans le ticket de réussite. La capture affichera la date, l’heure et un code de session unique.

Règles essentielles

Masqués
3 joueurs cachés. Chaque avis NPS à leur nom vaut 2 points.
Loyaux
Tous les autres joueurs. Chaque avis NPS à leur nom vaut 1 point.
Après un avis
Le joueur choisit soit sécuriser pour prendre +1 point garanti, soit tenter une épreuve.
Épreuve réussie
Si le mini-jeu est validé, le joueur prend +3 points. En cas d’échec : 0 point.
Enquêteur de la semaine
Le joueur avec le plus d’avis devient Enquêteur et obtient 1 tentative de démasquage.
Démasquage réussi
Si le bon Masqué est trouvé, le camp des Masqués perd 1 point. Sinon, rien ne se passe.

Choix après un avis

🛡️
Sécuriser
Gain garanti, sans risque. À utiliser si le joueur ne veut pas tenter sa chance.
+1 point sûr
⚔️
Tenter une épreuve
Accès à la salle des épreuves. Les jeux sont volontairement difficiles.
+3 si réussite / 0 si échec
