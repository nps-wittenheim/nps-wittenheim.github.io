Règles du jeu

Deux camps. Une concurrence directe. Un seul objectif : finir devant l’autre.
Les camps
Le jeu oppose deux camps : les Masqués et les Loyaux. Les Masqués sont moins nombreux mais plus puissants en points.
Les points
Un avis NPS positif obtenu à son nom rapporte : 2 points à un Masqué et 1 point à un Loyal.
Après un avis
Le joueur choisit entre : sécuriser pour prendre +1 point garanti, ou tenter une épreuve.
Les épreuves
Les épreuves durent environ 1 minute. Elles deviennent plus difficiles au fil du temps. Si l’objectif affiché est atteint, la victoire est validée.
Enquêteur de la semaine
Le joueur qui obtient le plus d’avis NPS positifs à son nom dans la semaine devient Enquêteur de la semaine.
Table ronde
Le lundi, l’Enquêteur peut envoyer un nom à la table ronde via WhatsApp. L’équipe peut se concerter, mais les Masqués sont dans le groupe.
Démasquage
Si le suspect envoyé est réellement un Masqué, le camp des Masqués perd 1 point. Sinon, aucun point n’est retiré.
Victoire finale
Le camp qui termine avec le plus de points gagne. Dans ce camp, le joueur ayant obtenu le plus d’avis positifs remporte le cadeau.
But du jeu : il faut que tout le monde comprenne vite qu’il y a une vraie concurrence entre le score total des Masqués et le score total des Loyaux.

Table ronde

Ouverte uniquement le lundi. L’Enquêteur de la semaine peut envoyer son accusation.
La règle du lundi
La table ronde n’est ouverte qu’une fois par semaine. La personne ayant obtenu le plus d’avis positifs à son nom peut voter. Elle peut se concerter avec l’équipe avant d’envoyer le nom du suspect, mais les Masqués sont présents dans le groupe et peuvent pousser sur une mauvaise piste.
